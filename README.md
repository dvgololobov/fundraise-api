# fundraise-api
Тестовый проект для Fundraise Up. REST API на базе koa и mongodb

## Настройка
Установите переменные окружения в файле .env:
#### Основной порт приложения
- PORT
#### Данные для подключения к БД Mongo
- DB_PORT
- DB_HOST
- DB_NAME

## Установка зависимостей
```bash
npm install
```

## Запуск приложения
```bash
npm run prod
```

# fundraise-api
Тестовый проект для Fundraise Up. REST API на базе koa и mongodb

## Настройка
Установите переменные окружения в файле .env
PORT - Основной порт приложения
Данные для подключения к БД Mongo
DB_PORT 
DB_HOST
DB_NAME

## Установка зависимостей
```
npm install
```

## Запуск приложения
```
npm run prod
```
